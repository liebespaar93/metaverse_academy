데이터 타입이란?

컴퓨터가 읽을 데이터의 형태를 말한다.\
메모리 주소를 접근하였을때 데이터 타입에 따라 알맞은 크기와 타입으로 읽어오게 설정 하는 것이다.

| 데이터 타입 | 설명 | 유형 |
|---|---|---|
| bool | 1과 0를 담는다 | ```true```, ```false``` |
| int | 정수를 담는다 | ```-2,147,483,648``` ~ ```2,147,483,647``` |
| float | 실수를 담는다 | ```-inf``` ~ ```inf``` |
| char | 문자를 담는다 | ```a```, ```b```, ```c``` ... |
