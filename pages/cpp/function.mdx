함수란?

함수는 일정한 규칙을 가지고 넣어준 값에 대한 동작을 실행한다.

```cpp
int main(int ac, char** av){
    return (0)
}
```

많이 보는 메인 함수이다.\
상위 함수를 보면 정수 타입의 ```ac``` 와 문자열 배열인 ```av``` 를 받는다.\
그리고 ```0```을 리턴하는 것을 볼 수 있다.

특정한 로직과 같은 반복되는 동작을 함수로 만들어서 실행 하는 것으로 작업량과 효율을 줄 일 수 있다.
