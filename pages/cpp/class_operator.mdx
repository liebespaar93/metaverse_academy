operator 란?

연산자를 받아 그것에 따른 작업을 만들어 주는 작업이다.

```=```, ```+```, ```-``` ... 와 같은 연산자들을 class에서 원하는 동작을 만드는 것이다

간단한 class \\ 만들어 보자

```cpp filename="soldier.hpp"
#include <iostream>

#ifndef SOLDIER_HPP
#define SOLDIER_HPP

class Soldier
{
public:
    char *name;
    int level;
    int speed;

    Soldier(const char *text, int level, int speed) {}

    Soldier(const Soldier &ref) {}

    ~Soldier(){};

    Soldier &operator=(const Soldier &ref) { return *this; }
    Soldier &operator+(const Soldier &ref) { /* 원하는 연산 */ return *this; }
    Soldier &operator-(const Soldier &ref) { /* 원하는 연산 */ return *this; }

    bool operator>(const Soldier &ref) { return this->level > ref.level; }
    bool operator<(const Soldier &ref) { return this->level < ref.level; }

    Soldier operator++()
    {
        Soldier temp(*this);
        this->level++;
        return temp;
    }
    Soldier &operator++(int)
    {
        this->level++;
        return *this;
    }
    // virtual void ShowName() = 0;
};

#endif
```

상위중 육안적으로 안보이는 사항은 전위 연산자와 후위연산자를 사용하는 것이다

> [!IMPORTANT]
>  전위 연산자의 경우 메서드에 (int)라는 인자가 보이는것을 볼수 있다

이러한 연산자들을 만들어 나만의 연산자 오버로딩을 생성하면 좀더 편한 클래스를 사용할 수 있다.
