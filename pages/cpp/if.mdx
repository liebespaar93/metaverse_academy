조건식

```1(true)``` 과 ```0(false)```으로 판별하여 어떠한 조건에 따라 실행할지 말지 정의를 한다


```cpp
if (나는빠박이다 == true) {
    std::cout << "머리가 반짝입니다." << std::endl;
}
```

이러한 식으로 조건을 비교하여 안에 있는 ```"머리가 반짝입니다."``` 가 실행 되는 것을 볼 수 있다.

또한 
```cpp
if (나는빠박이다 == true) {
    std::cout << "머리가 반짝입니다." << std::endl;
}
else if (진행중) {
    std::cout << "흑..." << std::endl;
} else {
    std::cout << "당신은 승리자 입니다." << std::endl;
}
```

이러한 식으로 여러가지 조건을 줄 수도 있다.
