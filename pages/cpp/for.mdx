반복문이란?

반복문은 일정한 규칙 아래 특정 작업을 반복하게 된다

반복문은 여러가지가 있다 (지금 생각나는 것만 작성)
| 명칭 | 설명 |
| --- |---|
| ```while``` | 조건이 true 일 경우 반복한다 |
| ```for``` | 특정한 규칙아래 반복을 한다 |
| ```foreach``` | 어떠한 배열의 안에있는 값을 반복하는 메서드 이다 |

```cpp
int i = 0
while ( i < 100)
{
    std::cout << "i : " << i << "값이 증가한다~!" << std::endl;
    i++;
}
```

i값이 99 까지 증가하는 것을 확인 할 수 있을 것이다.


```cpp

for (int i = 0;  i < 100; i++)
{
    std::cout << "i : " << i << "값이 증가한다~!" << std::endl;
}
```

상위 ```while```과 동작은 같지만 좀더 보기 편하게 작성할 수 있다.
