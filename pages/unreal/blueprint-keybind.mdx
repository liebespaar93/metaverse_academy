![unreal-blueprint-keybind-logic](./assets/unreal-blueprint-keybind-logic.png)

## Enhanced Input

최신 버전의 input 컨트롤 하는 방법이다.\
좀더 직관적이며 여러가지 기능을 한번에 사용할 수 있어 좋다

### Enhanced Input 생성

| 생성 | 아이템 목록 |
|---|---|
| ![unreal-blueprint-keybind-enhanced-create](./assets/unreal-blueprint-keybind-enhanced-create.png) | ![unreal-blueprint-keybind-enhanced-items](./assets/unreal-blueprint-keybind-enhanced-items.png) |

상위 보는 봐와 같이 ```mapping``` 과 ```input action``` 부분이 둘로 나누어 져 있다\
엑션을 먼저 만들고 멥핑에 들어가 엑션에 대한 keybind를 설정 해주어야 한다.

| input action | mapping |
|---|---|
| ![unreal-blueprint-keybind-enhanced-input-action](./assets/unreal-blueprint-keybind-enhanced-input-action.png) | ![unreal-blueprint-keybind-enhanced-input-mapping](./assets/unreal-blueprint-keybind-enhanced-input-mapping.png) |

상위 설정을 하고 이제 ```input``` 을 받고 싶은  ```actor```에 적용하면 된다._createMdxContent

### Enhanced Input 사용

| mapping | input action |
|---|---|
| ![unreal-blueprint-keybind-enhanced-use-mapping](./assets/unreal-blueprint-keybind-enhanced-use-mapping.gif) | ![unreal-blueprint-keybind-enhanced-use-action](./assets/unreal-blueprint-keybind-enhanced-use-action.gif) |

맵핑을 불러와 설정을 하면 이제 멥에 있는 ```input action```들을 사용 할 수 있게 된다.\
