## UObject

UClass들의 모든 정보를 기본 적인 데이터 CDO(Class Default Object)로 생성하여 하나씩 저장해 둔다.\
그리고 필요할 때 또는 새로 생성되었으 때마다 이미 만들어진 오브젝트를 clone(복사) 하는식으로 객체를 생성한다.
이러한 방식으로 생성자가 호출되서 동작하는 런타임을 줄일 수 있고 메모리를 그대로 전달만 하면 된다는 이점이 생긴다.

기본적인 상속 코드는 하위와 같다

```cpp
#pragma once

#include 'Object.h'
#include 'MyObject.generated.h' // 언리렇 라이브러리

UCLASS()
class MYPROJECT_API UMyObject : public UObject
{
    GENERATED_BODY()

};
```
